## 🚀 كيفية الاستخدام

### الطريقة الأولى: فتح مباشر
1. افتح ملف `cour.html` في أي متصفح ويب حديث
2. ابدأ الاختبار مباشرة

### الطريقة الثانية: خادم محلي
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx http-server

# أو باستخدام Live Server في VS Code
```

## 🎮 كيفية اللعب
1. **اختر الإجابة الصحيحة** من بين 4 خيارات
2. **استمع للصوت** لتأكيد الإجابة
3. **اقرأ الشرح** لفهم الإجابة الصحيحة
4. **تابع تقدمك** عبر الإحصائيات العلوية
5. **راجع أخطاءك** في نهاية الاختبار
6. **أعد الاختبار** بترتيب جديد للأسئلة
